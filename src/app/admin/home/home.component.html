Queue Simulator

This tool was created to demonstrate the power of our queue management system. Combining features like booking forms, queue capacities, access validation, late arrival management, and so on, we can automate queue management and drastically improve the processing speed at the access point.
All you need to do is to create your own scenarios (PARAMETERS) and the app will take care of generating the queues (QUEUE), simulating
the 24 hours flow (SIMULATION) and measuring performance (REPORTS).

Parameters

First, you need to provide a set of parameters which will be used to generate the queue and configure the way the system performs in real-life situations.

Queue

Based on these parameters the app will generate the individual queue for each tenant and after that the composite queue by combining the individual queues of each tenant.

Simulation

This step is where the app simulates the flow of trucks arriving at the access point and generates a short visual simulation and a log with the events, decisions and KPIs.

Reports

The reports section will display different KPIs and graphs emphasizing the performance of the system which can be used for further analysis or in your decision-making process.
